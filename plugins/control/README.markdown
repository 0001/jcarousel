jCarousel Control Plugin
========================

The jCarousel Control Plugin provides controls for jCarousel.

Getting started
---------------

To use the jCarousel Control Plugin, include the source file right after the jCarousel core file:

    <script type="text/javascript" src="http://ajax.googleapis.com/ajax/libs/jquery/1.6.4/jquery.min.js"></script>
    <script type="text/javascript" src="/path/to/jquery.jcarousel.min.js"></script>
    <script type="text/javascript" src="/path/to/jquery.jcarousel.control.min.js"></script>
    <link rel="stylesheet" type="text/css" href="/path/to/skin.css" />
